Why S+ and S− Are Conjugates of Each Other

This is a subtle but really important point about spinor representations of SO(2n). Let’s go through the logic in a clean, conceptual way.

1. What does “complex conjugate representation” mean?

A representation R is complex conjugate to R′ if there exists a matrix C such that

C−1 R*(g) C = R′(g)

for all group elements g.

So here, we want to understand how the complex conjugate of the spinor representation behaves.

2. Start from how spinors transform

You have:

ψ → exp[(i/4) ωij σij] P± ψ

So the representation matrices are:

R± = exp[(i/4) ωij σij] P±

3. Take the complex conjugate

Complex conjugation gives:

R±* = exp[−(i/4) ωij σij*] P±*

Now we want to relate this back to the original form using the charge conjugation matrix C.

4. Use the key identities

From the equations shown:

(i) For the generators:

C−1 σij* C = −σij

(ii) For chirality:

Cn−1 γF Cn = (−1)n γF

5. What happens to the projector?

The chiral projectors are:

P± = 1/2 (1 ± γF)

Taking complex conjugation and conjugating by C:

C−1 P±* C = 1/2 (1 ± (−1)n γF)

So:

  • If n is even, then (−1)n = +1, so P± → P±.
  • If n is odd, then (−1)n = −1, so P± → P∓.

6. Put it all together

Now apply C−1(·)C to the full representation:

C−1 R±* C = exp[(i/4) ωij σij] × (transformed projector)

For even n:

C−1 R±* C = R±

Each representation is equivalent to its own complex conjugate.

For odd n:

C−1 R±* C = R∓

The complex conjugate of S+ is S−, and vice versa.

7. Final interpretation

The only difference between S+ and S− is the chirality projector. Complex conjugation flips chirality when n is odd, but not when n is even.

Conclusion.
If n is even, then (S±)* ≅ S±.
If n is odd, then (S+)* ≅ S− and (S−)* ≅ S+.

8. Intuition

The whole effect comes from the sign in

C−1 γF C = (−1)n γF

This determines whether chirality is preserved or flipped under conjugation. Since S± are defined by chirality, that is exactly what determines whether they map to themselves or to each other.
